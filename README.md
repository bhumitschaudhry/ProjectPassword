# Password Generator

A simple, elegant password generator application built with HTML, CSS, and JavaScript.

## Features

- Generate random passwords with customizable length (4-32 characters)
- Include/exclude character types:
  - Uppercase letters (A-Z)
  - Lowercase letters (a-z)
  - Numbers (0-9)
  - Symbols (!@#$%^&*)
- Copy password to clipboard with one click
- Modern grayscale design with yellow accents
- Responsive design for mobile and desktop

## Usage

Simply open `index.html` in your web browser to start using the password generator.

1. Adjust the password length using the slider
2. Select which character types to include
3. Click "Generate Password" to create a new password
4. Click the copy icon to copy the password to your clipboard

## Technologies Used

- HTML5
- CSS3
- Vanilla JavaScript

No external dependencies required!
